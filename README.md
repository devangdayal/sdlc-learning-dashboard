# Engineering SDLC Mastery Dashboard

A lightweight personal learning dashboard for mastering the **entire Software Development Lifecycle (SDLC)** from fundamentals to production systems.

This tool helps track progress across core engineering domains including architecture, distributed systems, backend development, cloud infrastructure, observability, and performance engineering.

The dashboard is built as a **single HTML application** with local progress tracking and zero dependencies.


## Live Dashboard

[SDLC Learning Dashboard](https://devangdayal.github.io/sdlc-learning-dashboard/)



## Features

- Structured **multi-phase engineering roadmap**
- Track progress across **100+ engineering topics**
- **Searchable topic catalogue**
- **Phase-wise learning sections**
- **Progress bar with completion percentage**
- **Local progress persistence using browser storage**
- Fully **responsive layout**
- **Single-file HTML application**
- No backend required
